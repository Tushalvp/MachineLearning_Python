# Machine Learning Projects using Python
---
Creating Supervised and Un-supervised machine learning projects using Python 

### Project directory structure
------------ 
```
|    ├── README.md             <- Bird's eye view on the environment and project info (Your are Here!!).
|    |
|    ├── data                  <- Store data for analysis and making prediction.
|    |   │
|    │   ├── processed         <- The final, canonical data sets for modeling.
|    │   └── raw               <- The original, immutable data dump.
|    |
|    ├── function              <- Contains scripts, class objects and pre-defined functions  
|    |   │
|    |   ├── helper.py         <- Scripts to download or generate data, turn raw data into 
|    |   │                        features for modeling
|    |   │                        
|    |   └── visualization.py  <- Scripts to create exploratory and results oriented visualizations
|    │
|    ├── images                <-  jpg (or) PNG images to be used in reporting
|    │
|    ├── models                <-  Scripts to train models and then use trained models to   
|    |   │                         make  predictions
|    |   │                        
|    |   ├── projectname.IPYNB <-  Jupyter notebook to make predictions
```
